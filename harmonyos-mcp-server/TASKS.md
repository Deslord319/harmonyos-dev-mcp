## 🚫 已取消的功能

| 功能            | 取消原因                                                                                |
| --------------- | --------------------------------------------------------------------------------------- |
| **签名管理**    | HarmonyOS签名机制类似Apple，证书和Profile必须通过华为开发者门户在线申请，无法完全自动化 |
| **AGC API集成** | 需要开发者账号OAuth认证，涉及封闭的华为云服务，不适合MCP工具自动化                      |
